# Django Article Project

## Project Overview
This is a simple web application developed using the **Django framework**.  
The project displays articles about different cities and allows users to read detailed information about each article. The website also contains sections for **News, Events, and About**, making it similar to a small blog or information portal.

This project was created while learning **Full Stack Python at JSpiders**, where I practiced Django concepts like views, templates, dynamic URLs, and template inheritance.

---

## Features
- Display multiple articles on the home page
- Read full article details using dynamic URLs
- Separate pages for News, Events, and About
- Navigation bar for easy page navigation
- Card-based layout for articles
- Template inheritance using a base template
