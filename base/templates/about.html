About This Project

This project is a simple article web application developed using the Django framework. The main objective of this project is to understand the core concepts of Django such as views, templates, template inheritance, static files, and dynamic URLs.

This project was created as part of my learning journey in the Full Stack Python course at JSpiders. The guidance provided during the training helped me understand how Django can be used to build dynamic and structured web applications.

The purpose of building this project is to strengthen my practical knowledge of Django development and gain experience in creating real-world web applications using Python and the Django framework.
